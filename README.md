# ordneri_webassembly_learn
> 学习webassembly的一些示例
## 注意
- 本文给有一定nodejs基础的人阅读
- 作者能力有限，若有错误，欢迎指正
## 安装
- 安装nodejs
- 安装依赖
```batch
npm install
```
## 运行
- 在示例目录下运行`node run.js`+code/html中出现的网页的文件名（不包含.html,默认采用8881端口）
- 打开控制台中出现的链接
- 部分网页可能
- 运行示例
```batch
node run.js hello
```
- 注意
  - 项目中部分网页要在控制台运行相关函数，详见[教程](./note/webassembly.md)
